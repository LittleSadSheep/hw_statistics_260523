## 三、统计分析方法（至少三种）

### 方法一：独立样本 t 检验（性别消费差异）
- **H₀**：男女生日均消费均值无显著差异  
- **H₁**：男女生日均消费均值有显著差异  
- **检验**：Levene 方差齐性检验 → 选择 pooled/separate t-test  
- **显著性**：p < 0.05 拒绝 H₀

### 方法二：单因素方差分析 ANOVA（生源地/学院消费差异）
- **H₀**：不同生源地（或学院）学生日均消费均值相等  
- **H₁**：至少有一组均值不等  
- **事后检验**：若 ANOVA 显著，进行 Tukey HSD 多重比较  
- **显著性**：p < 0.05 拒绝 H₀

### 方法三：多元线性回归（影响食堂消费的因素）
- **因变量 Y**：月均食堂消费金额  
- **自变量 X**：性别、生源地、年级、口味评分、价格评分、服务评分、健康评分、环境评分、种类评分  
- **模型形式**：Y = β₀ + β₁X₁ + ... + β₉X₉ + ε  
- **诊断**：VIF 多重共线性检验、残差正态性、异方差检验（White/Breusch-Pagan）
